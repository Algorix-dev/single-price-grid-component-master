# Single Price Grid Component

A responsive pricing component built as part of a Frontend Mentor challenge. The project focuses on creating a clean, responsive card layout using semantic HTML and modern CSS techniques.

## 📖 Overview

This project recreates the provided Frontend Mentor design as accurately as possible. It adapts seamlessly across different screen sizes and includes an interactive hover effect for the call-to-action button on desktop devices.

### Features

* Responsive design for mobile and desktop screens
* Semantic HTML5 structure
* Modern CSS styling
* Hover effect on the **Sign Up** button
* Clean and accessible layout

## 🛠️ Built With

* HTML5
* CSS3
* Flexbox
